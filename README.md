## How to Install

1.  **Clone this repository:**

    ```bash
    git clone https://github.com/your-repo/fact-digi-landing-page.git
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd fact-digi-landing-page
    ```

3.  **Install dependencies:**

    ```bash
    pnpm install
    ```

4.  **Start the development server:**
    ```bash
    pnpm dev
    ```
